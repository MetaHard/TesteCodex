# Memória

Este arquivo é a fonte de referência para a criação de planilhas.
As IAs responsáveis **devem** consultar este conteúdo antes de gerar qualquer planilha.

## Diretrizes
- Sempre validar nomes de colunas com base neste documento.
- Preservar a ordem sugerida das colunas quando aplicável.
- Registrar observações relevantes no campo de notas da planilha.

## Estrutura padrão
1. Data de criação
2. Responsável
3. Descrição
4. Categoria
5. Observações
