# TesteCodex

Este projeto contém uma demonstração web que gera a base de planilhas a partir do arquivo `Memória.md`.

## Como rodar o site

1. Inicie o servidor:

   ```bash
   python app.py
   ```

2. Abra o navegador em `http://localhost:8000`.

## O que você verá

- As colunas sugeridas são extraídas do arquivo `Memória.md`.
- O conteúdo da memória é exibido para transparência.
